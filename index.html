О масштабируемой векторной графике(SVG)

Как правильно вставлять SVG

      Вставка картинкой.

                      
                <img
                src="picture.svg"
                alt="За стеклом">

      Фоновая картинка в CSS.

                      
                .picture {
                    background-image:
                        url(picture.svg);
                     }
    Этот способ подходит для оформительской графики, которой не нужно взаимодействие: фоны, иконки и другая мелочь.

      Третий способ, через object

      
                <object
                type="image/svg+xml"
                data="picture.svg">
                <img
                    src="picture.png">
                </object>

      Работают скрипты, взаимодействие, анимация — если они описаны внутри SVG.

      Между тегами object можно вставить фолбэк, который покажется, если браузер не говорит на SVG.

      Этот способ подходит, когда вам нужно вставить какую-то интерактивную графику: игрушки, графики и всякое сложное.

      Вставлять на страницу, как любые другие теги

      
                <h1>Квадрат</h1>
                <svg
                width="20"
                height="20">
                    <rect fill="#fc0"
                    width="20"
                    height="20"/>
                </svg>
