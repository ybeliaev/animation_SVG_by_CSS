О масштабируемой векторной графике(SVG)

Как правильно вставлять SVG

      Вставка картинкой.

                      
                <img
                src="picture.svg"
                alt="За стеклом">

      Фоновая картинка в CSS.

                      
                .picture {
                    background-image:
                        url(picture.svg);
                     }
    Этот способ подходит для оформительской графики, которой не нужно взаимодействие: фоны, иконки и другая мелочь.

      Третий способ, через object

      
                <object
                type="image/svg+xml"
                data="picture.svg">
                <img
                    src="picture.png">
                </object>
